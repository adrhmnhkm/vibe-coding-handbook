# Code of Conduct

Agar proyek ini nyaman untuk semua, kita menerapkan aturan dasar berikut:

1. Saling menghormati
   - Tidak menggunakan hinaan, serangan pribadi, atau bahasa kasar.
   - Kritik boleh, tapi serang idenya, bukan orangnya.

2. Inklusif untuk pemula
   - Jangan merendahkan orang yang masih belajar.
   - Bantu jelaskan jika ada yang bingung, bukan diketawain.

3. Tidak ada konten berbahaya
   - Jangan sengaja membagikan cara menyalahgunakan kredensial, melakukan serangan, atau aktivitas ilegal.
   - Kalau membahas security, fokus pada edukasi dan mitigasi.

4. Tidak ada data sensitif
   - Jangan mengupload contoh yang berisi data pribadi nyata atau kredensial asli.

5. Diskusi sehat
   - Berusaha asumsikan niat baik orang lain.
   - Jika ada konflik, utamakan penyelesaian dengan tenang.

Pelanggaran berat (misalnya pelecehan, ujaran kebencian, spam berbahaya) bisa berakibat pada pemblokiran dari proyek ini.
